PDFWidgetAnnotationView Class Reference

Inherits from UIView
Declared in PDFWidgetAnnotationView.h

Overview

The PDFWidgetAnnotationView represents a subview of a PDFView that represents an interactive or accessory element. A PDFForm is an example.

fontSizeForRect:value:multiline:choice:

For widget annotations with text, this method returns how font size should be initially scaled with respect to the bounding rect.

Parameters

The bounding frame of the widget in iOS screen space, not PDF canvas space.

The text to render.

YES if multiple lines of text are permitted.
